### Технический подход
- Использование CSS переменных для обеспечения консистентности.
- Tailwind CSS классы будут заменены на кастомные переменные (например, `bg-[var(--red-primary)]`).
- Приведение всего UI к стандарту "минималистичный премиум-стартап" (Apple-like).
