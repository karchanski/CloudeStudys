# Инструкция по запуску CloudeStudy на Windows

Для запуска проекта CloudeStudy на Windows рекомендуется использовать Docker, так как он обеспечивает одинаковую среду выполнения для всех компонентов.

## Предварительные требования

1. **Установите Docker Desktop**:
   - Скачайте и установите [Docker Desktop для Windows](https://www.docker.com/products/docker-desktop/).
   - Убедитесь, что после установки Docker запущен.
2. **Установите Git**:
   - Скачайте и установите [Git for Windows](https://git-scm.com/).
3. **Клонируйте репозиторий**:
   - Откройте PowerShell или терминал.
   - Выполните команду:
     ```powershell
     git clone https://github.com/karchanski/cloudestudy.git
     cd cloudestudy
     ```

## Запуск проекта

В проекте есть файл `docker-compose.yml`, который упрощает запуск всех сервисов (Backend, Frontend, Telegram Bot).

1. **Запуск контейнеров**:
   - В терминале, находясь в папке проекта, выполните команду:
     ```powershell
     docker-compose up --build
     ```
   - При первом запуске Docker скачает необходимые образы и соберет проект. Это может занять некоторое время.

2. **Доступ к приложениям**:
   - После того как все контейнеры запустятся, приложения будут доступны по стандартным портам (проверьте настройки в `docker-compose.yml` или конфигурационных файлах).

## Решение проблем

- **Docker не запускается**: Убедитесь, что у вас включена виртуализация (Hyper-V или WSL 2) в BIOS/UEFI.
- **Ошибки при сборке**: Попробуйте остановить все контейнеры командой `docker-compose down` и запустить заново `docker-compose up --build`.
- **Порты заняты**: Если какие-то порты уже заняты другими приложениями, отредактируйте файл `docker-compose.yml` и измените маппинг портов (например, с `80:80` на `8080:80`).
